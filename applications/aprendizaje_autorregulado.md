## 1. Modelo Base: Zimmerman (2000)

El SIAAE se sustenta en el modelo cíclico de Zimmerman, que describe la autorregulación del aprendizaje en tres fases interdependientes:

```
┌─────────────────────────────────────────────────────────┐
│                  CICLO ZIMMERMAN                        │
│                                                         │
│   ┌──────────┐     ┌──────────┐     ┌───────────────┐  │
│   │PREVISIÓN │────▶│EJECUCIÓN │────▶│AUTORREFLEXIÓN │  │
│   │          │     │          │     │               │  │
│   │• Metas   │     │• Atención│     │• Autojuicio   │  │
│   │• Planif. │     │• Estrat. │     │• Autorreacción│  │
│   │• Autoef. │     │• Autoobs.│     │• Atribución   │  │
│   └──────────┘     └──────────┘     └───────────────┘  │
│         ▲                                    │          │
│         └────────────────────────────────────┘          │
└─────────────────────────────────────────────────────────┘
```

## 2. Sustento por Fase — Autores Integrados

### Fase 1: Previsión
| Componente | Autor Clave | Aplicación Operativa |
|---|---|---|
| Establecimiento de metas | Bandura | Metas específicas, proximales y desafiantes |
| Autoeficacia académica | Bandura | Medición ACAES en ingreso |
| Orientación motivacional | Frankl | Sentido vocacional como ancla motivacional |
| Planificación estratégica | Vygotsky | Andamiaje inicial del plan de estudios |

### Fase 2: Ejecución/Control volicional
| Componente | Autor Clave | Aplicación Operativa |
|---|---|---|
| Regulación emocional | Linehan / Siegel | Talleres DBT-adaptados, mindfulness |
| Clima de aprendizaje | Rogers | Vínculo pedagógico, feedback apreciativo |
| Aprendizaje colaborativo | Vygotsky | Programa tutores pares (ZDP) |
| Atención y concentración | Siegel | Técnicas de ventana de tolerancia |

### Fase 3: Autorreflexión
| Componente | Autor Clave | Aplicación Operativa |
|---|---|---|
| Atribución causal adaptativa | Beck | Reestructuración cognitiva post-evaluación |
| Autojuicio | Bandura | Calibración de autoeficacia real vs. percibida |
| Satisfacción/reacción | Rogers | Autoconcepto académico positivo |

## 3. Indicadores Cuantificables SIAAE

### Variables de entrada (T0 — Diagnóstico inicial)
- Autoeficacia académica (ACAES o MSLQ)
- Regulación emocional (DERS-16)
- Estrategias de aprendizaje (MSLQ completo)
- Síntomas ansiedad/depresión (DASS-21)
- Sentido de pertenencia (escala ad hoc)

### Variables de proceso (T1, T2 — Seguimiento)
- Asistencia a actividades psicoeducativas
- Participación en tutoría pares
- Solicitudes de acompañamiento
- NPS de satisfacción con servicios

### Variables de resultado (T3 — Final)
- Promedio académico delta (ingreso vs. cierre)
- Retención semestral
- Tasa de aprobación primer año
- Reinscripción semestre siguiente
