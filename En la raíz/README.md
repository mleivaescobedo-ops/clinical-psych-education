## 📌 Propósito

Este repositorio integra los grandes autores de la psicología clínica con aplicaciones concretas en **bienestar estudiantil**, **aprendizaje autorregulado** y **salud mental en educación superior**, con foco en el contexto chileno.

Sirve como:
- Base teórica para el **SIAAE** (Sistema Integral de Aprendizaje Autorregulado)
- Insumo para investigación aplicada y postulaciones ANID
- Recurso de onboarding para equipos de bienestar
- Portfolio académico sistematizado
